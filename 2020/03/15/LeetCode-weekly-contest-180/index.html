1380. Lucky Numbers in a Matrix

统计每行的最小值所在的列数和每列最大值所在的行数，判断是否有相等的。

时间复杂度: O(m * n),
空间复杂度: O(m + n).

1381. Design a Stack With Increment Operation

用数组实现栈即可。

时间复杂度:

  • Constructor: O(1)
  • push: O(1)
  • pop: O(1)
  • increment: O(k)
    空间复杂度: O(maxSize)

还有一种Lazy Increment的算法，increment时，只做一个标记，只有到pop的时候才真的增加。可以实现increment的O(1).

1382. Balance a Binary Search Tree

先用中序遍历将数据排序，然后再使用二分法重新建树。

时间复杂度: O(N),
空间复杂度: O(height).

1383. Maximum Performance of a Team

贪心的算法。先按照efficiency从大到小排序，挑选效率最高的k个人，然后再遍历效率稍低的人，尝试把他们加入队伍(剔除掉speed最小的), 更新可能的Maximum performance.

时间复杂度: O(N log N + N log K),
空间复杂度: O(N + K).

相似的题目: LeetCode 857.
